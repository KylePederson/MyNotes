Git
Command Description
git init Create .git repository in working directory
git add filename Add file to staging area
git status show the status of modified items in staging area and/or working directory
git commit -m "message" Create commit for files in staging area
git log Show previous commits
git rm --cached -r . Remove files from staging area
git remote add origin https://... Add remote repository address
git push -u origin master Push latest commits to remote repository
git branch branchname Create a new branch in working directory
git branch Show branch(s) in working directory
git checkout branchname Change to new branch
