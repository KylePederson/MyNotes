String Commands
Term Syntax Description
CONCAT SELECT CONCAT ( column_1, "-", column_2)
FROM tablename; 		concatenates multiple specified values
CONCAT_WS SELECT CONCAT_WS ("-", column_1, column_2)
FROM tablename; 		With Separator - concatenates multiple values together with a separator
SUBSTRING SELECT SUBSTRING ('Hello World', 1, 5);
  • ('Hello World', 1, 5) - Extracts a portion of the string by start location and number of characters
  • ('Hello World', 7) - Extracts a portion of the string by start location and going to the end of the string
  • ('Hello World', -3) - Extracts a portion of the string starting from the last character and going backward
REPLACE SELECT REPLACE ('Hello World', 'l', '7'); replaces the specified value with a new value
CHAR_LENGTH SELECT CHAR_LENGTH (column_1)
FROM tablename; 		returns the number of characters in a string
UPPER SELECT UPPER (column_1)
FROM tablename; 		converts all characters to uppercase
LOWER SELECT LOWER (column_1)
FROM tablename; 		converts all characters to lowercase
DISTINCT SELECT DISTINCT (column_1)
FROM tablename; 		Returns only the unique values for the specified column
ORDER BY SELECT (column_1, column_2, column_3)
FROM tablename
ORDER BY column_2;
  • ORDER BY column_2: Orders the results based on specified column
  • ORDER BY 2: # value specifies the column to order by
  • ORDER BY 2 DESC: sorts in descending order
  • ORDER BY 2 ASC: sorts in ascending order
LIMIT SELECT column_1
FROM tablename LIMIT 10; 		Specifies the max number of results to return
LIKE SELECT column_1, column_2
FROM tablename
WHERE column_1 LIKE '%da%';
  • % : Wildcard
  • _ : Wildcard to find any ONE character
  • \ : Nullifies the SQL effects of the next character
