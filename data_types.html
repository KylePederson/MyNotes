Data Types
Term Syntax Description
VARCHAR VARCHAR(size) Variable length string - Size indicates max characters from 0 to 65535
CHAR CHAR(size) Fixed length string - Size indicates max characters from 0 to 255
INT INT(size) A medium integer. Signed Range -2147483648 to 2147483647. Unsigned 0 to 4294967295.
DECIMAL DECIMAL(total digits, digits after decimal) always adds a decimal
FLOAT FLOAT reliable to 7 digits
DOUBLE DOUBLE reliable to 14 digits
DATE DATE YYYY-MM-DD
TIME TIME HH:MM:SS
DATETIME DATETIME YYYY-MM-DD HH:MM:SS
NOW NOW()
TIMESTAMP TIMESTAMP YYYY-MM-DD HH:MM:SS
